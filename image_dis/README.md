## 소개

해당 파일은 이미지를 불러들어 원하는 포인트에 점을 찍어 그 거리를 알수 잇게 해줍니다 

 Wd는 95mm로 설정하엿고
 이미지는 고래밥이미지를 대상으로 측정하여 픽셀크기를 계산하엿습니다
## 설치
